# LaTeX Whiteboard LaTeX whiteboard is a tool for creating markdown-enabled LaTeX boards. ## Quick start - To get started, fork this repl - Edit this file, but leave the first line (beginning with a `script` tag) intact. - Run the repl! You should see this document rendered in the output pane. - As you type in this file, your edits show up live. - You can share the link to the output with your students or friends and they can follow along ## Credits This makes use of [`susam/texme`](https://github.com/susam/texme) and is a remix of [muboard](https://github.com/susam/muboard) by the same author.
